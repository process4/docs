8 if you use Windows Vista, Windows 7, Windows Server 2008:

or

### Solution

Open up the Internet Explorer browser and click on the following
options, tabs, or buttons:

***Tools*** - ***Internet Options***  
Security tab for "***Internet***"  
Custom Level... button
